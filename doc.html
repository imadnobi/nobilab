nobiLAB : Documentation

Simple Programming languange for optimization modelling with various solver from exact, heuristic, and metaheuristics.
Developed by Nobita and friends from 22nd century, Indonesia.
START nobiLAB


nobiLAB contains 5 main part:
1. Objective Parameter (OP). This is paremeters that desire to be optimized. Each parameter has their range {min:max}
2. Objective Function (OF). This is a formula to calculate an objective function. That is a function that means the PURPOSE of optimization. In definition, optimization problem simply "What is the best value for parameters (OP) as result minimal/maximal objective function (OF)".
3. Constraint (constr). Constraint is a rule and boundary of problem definition.
4. Supporting variable. Variables which need for calculating OF.
5. Supporting code. Code or extra matematical operation for calculating OF.
